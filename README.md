This is a trajectory planning project using receding horizon style(or MPC, if you are used to call it) using global optimization method based on IGO and GMM as searching distribution. 
Written by the first author (well, thankful to AI's help) of MGIGO-MPC, it's easy to figure out the algorithm given by the article. However, as the first author is not good at
writing engineering codes, it's only an initial trial that verifing the correctness of the algorithm (Numerically finding whether the results are correct).
These codes are just for trials as numerical examples that try to verify whether a cost function is correct. Also, the codes included in GMM_IGO are solvers equipped with different types.

If you want to figure out the algorithm in the article MGIGO-MPC you can just use these codes. For practicle engineering uses, see my friend's projects(Lipeng Qi GMM-IGO). He turned some examples and the solver into CUDA version.
